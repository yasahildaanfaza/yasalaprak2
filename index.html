HTML ⯈

HTML, singkatan dari Hypertext Markup Language, merupakan bahasa pemrograman fundamental yang digunakan untuk menciptakan halaman situs web. Salah satu aspek penting dari HTML adalah penggunaan tag atau tanda, yang digunakan untuk menentukan struktur dan konten dari sebuah halaman web.Tag HTML merupakan sebuah tanda awal dan akhiran dari sebuah elemen. Tag dibuat dengan kurung < > dan berisikan nama tag dan terkadang ditambahkan dengan atribut misalnya seperti < c >, < body >, dan lainnya. Setiap tag HTML memiliki fungsi yang unik dan dapat memengaruhi cara sebuah halaman web ditampilkan dan berinteraksi dengan pengguna. Oleh karena itu, penting bagi para pengembang web untuk memahami fungsi-fungsi tag HTML ini agar mereka dapat membuat halaman web yang baik dan berfungsi dengan baik.

CSS ⯈

Css adalah singkatan dari Cascading Style Sheet yang dipergunakan untuk menentukan tampilan dan format pada sebuah website. Artinya, dengan adanya CSS bisa mengatur mulai dari font, warna tulisan dan latar belakang pada website tersebut. Cascading Style Sheet (CSS) bisa diintegrasikan langsung ke dalam file HTML atau menggunakan file terpisah. Dengan adanya alat tersebut bisa mempersingkat waktu dan cukup menuliskan kode satu kali saja.

JavaScript ⯈

JavaScript adalah bahasa pemrograman yang digunakan dalam pengembangan website agar lebih dinamis dan interaktif. Java Scipt berfungsi untuk membuat elemen yang mampu meningkatkan interaksi pengunjung seperti menu drop-down, animasi, dan warna background dinamis. Bahasa pemrograman ini memungkinkan pengunjung berinteraksi dengan halaman web, dan Anda pun bisa menyajikan user experience yang lebih baik.
